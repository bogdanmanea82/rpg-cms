Docker Compose Learning Environment

This page is served by nginx running in a Docker container.

The container was started using Docker Compose, which orchestrates multiple services defined in docker-compose.yml.

This demonstrates how Docker Compose simplifies managing multi-container applications.

Live Updates

Because this directory is mounted as a volume, changes to files appear immediately without restarting the container.

This is how you'll develop with Docker - your code lives on your host, mounted into containers for execution.
